# vite-react-tsx-library

React library boilerplate with Vite to publish on npm
Added Ts support

## To run in development

```bash
npm run dev
```

## To run tests

```bash
npm test
```

or

```bash
npm run watch
```

## To build and publish

```bash
npm run build
```

followed by

```bash
npm publish
```
